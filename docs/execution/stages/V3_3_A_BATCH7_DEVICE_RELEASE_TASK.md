# V3.3-A / A7 真机验收与交付任务

## 真机主流程

1. 旧普通用户登录并完成现有需求/旧物基础流程；
2. 创建设计师或工程师身份；
3. 提交认证并查看身份与认证分离状态；
4. 创建组织、发邀请、接受邀请；
5. 切换个人和组织工作台；
6. 创建/进入项目并邀请 viewer；
7. 验证 viewer 只读；
8. 停职或移除成员并验证页面、API、文件和消息权限回收；
9. 演示后台职责分离。

## 交付物

- 阶段执行报告；
- 数据迁移与备份说明；
- 越权测试摘要；
- 真机验收证据；
- 兼容字段和未决问题；
- 干净源码包；
- 是否允许进入 V3.3-B 的明确结论。
