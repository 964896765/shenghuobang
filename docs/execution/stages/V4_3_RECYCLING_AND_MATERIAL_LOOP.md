# V4.3 回收与材料闭环

## 1. 用户可见结果

用户可从产品档案或回收入口发起回收，获得初估和附近回收商报价，完成复检、最终价格、交接和结算，并获得回收去向证明。专业回收组织可以记录拆解、可复用零件和再生材料批次。

## 2. 流程

产品实例 → 回收申请 → 初步估价 → 回收商报价 → 上门/邮寄 → 实物复检 → 最终报价确认 → 交接 → 结算 → 拆解 → 零件分流 → 材料称重/纯度 → 再生材料批次 → 去向/新产品关联

## 3. 数据增量

- 回收任务和上门调度
- 实物复检及报价版本
- 回收交接凭证和所有权终止/转移
- 拆解批次、拆解行和损耗
- 可复用零部件实例
- 材料分类、重量、纯度和检测
- 再生材料批次
- 材料去向、销售/领用和新产品关联
- 回收证明和环保统计

## 4. 权限和隐私

- 回收员只能查看被分配任务和必要联系信息。
- 回收商不能查看历史所有者、原始购买价和无关维修隐私。
- 拆解企业只能看已交接给本组织的物品。
- 材料去向修改必须追加更正记录和审计。
- 重量、纯度和结算数据需要双人复核或证据附件。

## 5. 验收门禁

- 回收交接后所有权和物品状态一致。
- 最终价格必须由用户确认或进入争议流程。
- 拆解产出与投入质量守恒差异在允许损耗范围内并可解释。
- 零件和材料批次来源可反向追踪到产品实例。
- 再生材料进入新产品时形成闭环关联。
